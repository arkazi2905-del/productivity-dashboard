#  Productivity Dashboard

A responsive and interactive Productivity Dashboard built using HTML, CSS, and JavaScript.  
It includes task management features, authentication system, and form validation with a modern UI design.


## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage (for saving data)


## ✨ Features
- 🔐 Login & Signup system
- 📝 Add, edit, delete, and complete tasks
- 🔍 Filter tasks (All / Completed / Pending)
- 📅 Due date and category selection
- 📧 Contact form with validation
- ⚡ Real-time email and password validation
- 🌙 Dark mode toggle
- 🕒 Live digital clock with greeting message
- 📱 Fully responsive design


## 🚀 How to Run
1. Clone or download the project  
2. Open `index.html` in a browser  
3. Start using the application
